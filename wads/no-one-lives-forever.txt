Title                   : No One Lives Forever!
Filename                : no-one-lives-forever.wad
Release date            : 12-17
Author                  : El Tupi
Email Address           : drhodd@gmail.com
Other Files By Author   : Castle Argh, Precious Snowflake

Description             : Consists of a building and an outside area. Inside
                          the building is a 4 story killing ground that makes
                          use of slopes and 3D floors. Run outside to get
                          some health and prepare to get back in the action.

                          Contains all weapons excpet the BFG. 

                          As stated elsewhere this map requires ZDoom or
                          GZDoom and while this map might run on other
                          engines it will NOT WORK without the bridges, ramps
                          or the ability to jump.
